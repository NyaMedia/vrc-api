# Nya Media API for VRChat prefabs

See [vrc.nya.llc/spec](https://vrc.nya.llc/spec) for API docs. You can use the public API!

Working prefab implementation can be found at [artifishvr/MediaUI](https://github.com/artifishvr/MediaUI)

If you'd like to make your own prefab implementation, the spec & code above should be enough to get you started as long as you know U#, but you can also reach out on the [MediaUI discord server](https://mediaui.vrc.bz) for help! Keep in mind you'll need basic programming knowledge/familiarity with U#. If you don't, you can always use the existing prefab linked above.

**DO NOT** use this api for anything other than VRChat prefabs without permission from Nya Media.

## The Code

This code is open source to follow after the inspiration, [lamp/vrchat-youtube-search-api](https://gitea.moe/lamp/vrchat-youtube-search-api), but the code is *really* suboptimal and hard to read as it has to play within VRChat's limits, so don't take this as my best work.

Feel free to open Issues and PRs if you want to contribute or have suggestions for improvements!

## Self-hosting

Nya Media's API requires an API key for use, which is a closed service. Reach out via Bluesky, Twitter, or Email if you want access.

It's likely you could bypass the API key requirement, but you'll probably get blocked either intentionally or unintentionally.
